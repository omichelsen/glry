# glry-search-api

Backend API for glry-search.

## Endpoints

| Endpoint                                | Methods
| --------                                | -------
| /cartoons                               | GET
| /cartoons/tags                          | GET
| /cartoons/tags/merge                    | POST
| /cartoons/tags/`:id`/tags               | POST
| /cartoons/tags/`:id`/tags/`:tag`        | DELETE
